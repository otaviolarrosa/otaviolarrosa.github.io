Working with correlationId and Serilog in dotnet core

When working with distributed applications, we need to take care about rastreability. An efficient way to deal with rastreability is to implement correlationId to our requests and workers.
To implement this concept, we'll make use of an API that can be found at otaviolarrosa/DotnetApi (github.com). First, we'll need to install the following packages on solution:
1 - Serilog
2 - Serilog.AspNetCore
3 - Serilog.Enrichers.CorrelationId
